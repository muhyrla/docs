## Pro trial

Every Discord server gets a one-time **14-day Pro trial**. It's offered to you automatically — no code needed — and can be claimed once per server. During the trial your workspace has Pro limits: 500 monthly credits and 20 knowledge sources.

## Promo codes

A promo code grants your workspace a specific plan (Pro or Team) for a fixed number of days.

To redeem one, open the **Billing** page, enter the code, and confirm. The plan applies immediately.

Rules worth knowing:

- A workspace can redeem a given code **once**.
- Codes have a limited number of total activations — a code that worked for someone else may already be used up.
- Redeeming a code for the plan you already have **adds its days** to your remaining time.
- If you redeem a code for a *higher* plan than your current grant, the higher plan takes effect right away; the remaining days of the lower one wait and resume afterwards. You never lose paid days.

Along with the plan, a code brings the matching credit allowance for its duration — those credits expire when the granted plan ends.
